Kiri:Moto is an extensible, multi-purpose slicing and visualization engine that produces output for:

  • : 3 axis CNC toolpaths
  • : GCode for 3D printers
  • : DXG / SVG cut paths

    Application access

  • -- Native web app
  • -- in Onshape App Store
  • -- as a Thing App

    File Load and Import

  • Drag and drop onto the build grid
  • Or load with Function [Import]
  • Cached file access with Function [+]

    Start Slicing

  • Select device mode (FDM, CAM, Laser)
  • Select device profile under Devices
  • Set slicing parameters in right menu
  • Test slicing settings with Function [Slice]
  • Preview slicing output with Function [Preview]
  • Export gcode with Function [Export]

    Refer to the Grid.Space Wiki for more information
