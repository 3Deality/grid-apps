Help & Introduction

Kiri:Moto is an extensible, multi-purpose slicing
and visualization engine that produces output for

  • : GCode for 3D printers
  • : 3 Axis CNC toolpaths
  • : DXG / SVG cut paths

    Learn More

  • Watch a tutorial on YouTube
  • Follow Grid.Space on Twitter
  • Join the user group on Facebook

    File Load & Import

  • Drag and drop an STL onto the grid
  • Or load with Function [Import]

    Start Working

  • Select device mode: FDM, CAM, Laser
  • Select device profile under Devices
  • Set slicing parameters in right menu
  • Test slicing settings with Function [Slice]
  • Preview slicing output with Function [Preview]
  • Export gcode with Function [Export]
